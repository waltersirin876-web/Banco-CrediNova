¡Bienvenido a la plataforma digital de **Banco-CrediNova**! Este es un simulador de banca privada diseñado para ofrecer una experiencia premium y segura.

## ✨ Características
* **Gestión de Saldo:** Depósitos y transferencias en tiempo real.
* **Bóveda de Seguridad:** Función para ocultar el saldo con un clic.
* **Simulador de Préstamos:** Cálculo automático de cuotas mensuales.
* **Generador de QR:** Cobros rápidos mediante códigos QR dinámicos.
* **Modo Inteligente:** Cambio automático entre modo claro y oscuro.
* **Estado de Cuenta:** Generación de reportes en PDF.

## 🛠️ Tecnologías
* **HTML5 / CSS3:** Diseño responsivo y temas dinámicos.
* **JavaScript (ES6):** Lógica financiera y persistencia en `localStorage`.
* **html2pdf.js:** Para la exportación de documentos.
* **QRCode.js:** Para la generación de códigos de cobro.
